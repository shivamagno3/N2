## 🚀 Deploy to GitHub Pages (Get a Live Link)

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a new repository
1. Click **"New"** or go to `github.com/new`
2. Name it: `nitya-birthday`
3. Set it to **Public**
4. Click **"Create repository"**

### Step 3 — Upload your files
1. On the repository page click **"uploading an existing file"**
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `img1.jpg`, `img2.jpg`, `img3.jpg`, `img4.jpg` (your photos)
   - `music.mp3` (your romantic song)
3. Click **"Commit changes"**

### Step 4 — Enable GitHub Pages
1. Go to your repo → **Settings** tab
2. Scroll to **"Pages"** in the left sidebar
3. Under **"Branch"**, select `main` → `/ (root)` → click **Save**
4. Wait ~1 minute

### Step 5 — Get your live link 🎉
Your site will be live at:
```
https://YOUR-GITHUB-USERNAME.github.io/nitya-birthday/
```
